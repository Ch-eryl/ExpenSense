# Spending Personality Detector

A fintech analytics tool that analyzes bank transaction data and identifies spending behavior.

Users upload a CSV of transactions and receive:

- Spending personality
- Monthly burn rate
- Savings insights
- Financial recommendations

Tech Stack
- Python
- Flask
- Pandas
- Simple behavioral scoring model

Future Improvements
- AI financial advisor
- Interactive charts
- Bank API integrations

Demo
Upload a CSV and instantly understand your spending habits.
